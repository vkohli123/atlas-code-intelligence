# Atlas Branding

**Name:** Atlas  
**Full name:** Atlas — AI Codebase Intelligence & PR Review Agent  
**Tagline:** Navigate any codebase with source-grounded AI.

## Why Atlas?

Atlas is a map for complex engineering systems. It turns a large repository into a navigable knowledge graph of files, functions, architectural flows, risks, and pull-request changes.

The metaphor is deliberate: Atlas does not just answer questions. It helps engineers orient themselves inside unfamiliar systems, trace dependencies, understand design trade-offs, and review changes with precise source citations.

## Positioning

Atlas is not a chatbot. It is a code intelligence system combining repository indexing, AST-aware chunking, hybrid retrieval, grounded generation, evaluation metrics, and PR risk analysis.

## Resume line

Built **Atlas**, an AI codebase intelligence and PR review platform indexing 100K+ LOC with AST-aware chunking, hybrid retrieval, reranking-ready search, source-grounded answers, and hunk-level review findings.
